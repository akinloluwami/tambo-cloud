---
title: useTamboComponentState
description: Persist component-local state that the AI can reference.
---

<Callout type="info" title="Counter Example">
  `useTamboComponentState` mirrors React’s `useState` but stores the value in the
  conversation so Tambo can reason about it in later replies.

```tsx title="Counter.tsx"
"use client";

import { useTamboComponentState } from "@tambo-ai/react";

export default function Counter() {
  const [count, setCount] = useTamboComponentState<number>("count", 0);

  return <button onClick={() => setCount(count + 1)}>Clicks: {count}</button>;
}
```

</Callout>

## Returned Values

- **value** – current state value.
- **setValue** – mutate and sync value.
