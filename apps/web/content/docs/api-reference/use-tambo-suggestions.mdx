---
title: useTamboSuggestions
description: Manage AI-generated suggestions for the next user message.
---

`useTamboSuggestions` provides a simple API for surfacing and accepting
next-action suggestions generated by Tambo.

```tsx title="SuggestionList.tsx"
"use client";

import { useTamboSuggestions } from "@tambo-ai/react";

export function SuggestionList() {
  const {
    suggestions,
    selectedSuggestionId,
    accept,
    generateResult,
    isPending,
  } = useTamboSuggestions();

  if (isPending) return <p>Loading suggestions…</p>;

  return (
    <div>
      <button onClick={() => generateResult.mutate()} className="btn-sm mb-2">
        Refresh
      </button>

      {suggestions?.map((s) => (
        <button
          key={s.id}
          className={`block w-full text-left rounded px-3 py-2 hover:bg-muted ${
            selectedSuggestionId === s.id ? "bg-primary/10" : ""
          }`}
          onClick={() => accept(s.id, { submit: true })}
        >
          {s.text}
        </button>
      )) ?? <p>No suggestions yet.</p>}
    </div>
  );
}
```

## Workflow

1. Click **Refresh** to fetch new suggestions via `generateResult.mutate()`.
2. Click a suggestion to accept and immediately submit it (`accept(id, { submit: true })`).

## Flags

- **acceptResult / generateResult** – mutation objects from React Query.
- **isPending / isSuccess / isError / error** – derived convenience booleans.
