---
title: useTamboClient
description: Direct access to the underlying Tambo client instance.
---

Need to drop down and call a low-level method? Grab the raw client with
`useTamboClient()`.

```ts title="advanced.ts"
const { client } = useTamboClient();

await client.someAdvancedMethod();
```
